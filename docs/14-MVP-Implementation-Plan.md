# Success Criteria

The judge should experience the following flow in under five minutes:

1. Sign in
2. Complete AI onboarding
3. Watch LearnOS generate a personalized journey
4. View the dashboard
5. Start today's learning session
6. Ask the AI Mentor a contextual question
7. Complete a quiz and assignment
8. Watch the roadmap adapt
9. See progress update automatically

If these nine steps feel smooth, intelligent, and polished, the MVP is successful.
